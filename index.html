Sort your arrays using Rust

SortLib is a rust crate and wasm package that allows you to sort your arrays fast and easily. Built using Rust, this library enables efficient and safe concurrency for both native runtimes and in the browser.

Love the performance of your app

Sorting for single threaded sorts is as fast as native sorting speeds. Multi threaded sorts are significantly faster for large datasets. Sorting 100k random values can be up to 42% faster when parallelized with only 2 workers!

Optimized for usability

Parallel algorithms scale to low data volume and have no performance overhead from threading when the data is sufficiently small. Never worry about handling performance for different data sizes.

Easy to get started

Simple APIs with easy installs make it painless to get started. Just import the package or crate and start sorting your data.
